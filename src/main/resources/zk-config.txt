(4).lcp_framework只存放lcp平台相关的参数，全部加载在JVM中，定时刷新，所以可以放到一个资源占用很小的db中.
/lcp/dev/mysql/master/lcp_framework { "url": "jdbc:mysql://123.57.204.187:3306/lcp?useUnicode=true&characterEncoding=utf-8", "userName": "root", "userName": "root", "password":"111111", "driverClassName":"com.mysql.jdbc.Driver"}

(5).lcp_framework提供各种功能性插件,由于数据库名称的不同，plugIn涉及到的DAO有可能需要修改@DAO注解中的catalog为实际数据库

/lcp/dev/mysql/master/lcp_framework/plugin/passport { "url": "jdbc:mysql://123.57.204.187:3306/lcp_passport?useUnicode=true&characterEncoding=utf-8", "userName": "root", "userName": "root", "password":"111111", "driverClassName":"com.mysql.jdbc.Driver"}
